Quality (or Vapor Quality) and Two-Phase Mixture Properties
\(\displaystyle x = \frac{m_{\text{g}}}{m_{\text{f}} + m_{\text{g}}}\)
\(\displaystyle v = (1-x)v_f + xv_g = v_f + x(v_g - v_f)\)
\(\displaystyle u = (1-x)u_f + xu_g = u_f + x(u_g - u_f)\)
\(\displaystyle h = (1-x)h_f + xh_g = h_f + x(h_g - h_f)\)
\(\displaystyle s = (1-x)s_f + xs_g = s_f + x(s_g - s_f)\)
Symbols
  • \(x\) — quality (or vapor quality)
  • \(m\) — mass
  • \(v\) — specific volume
  • \(u\) — specific internal energy
  • \(h\) — specific enthalpy
  • \(s\) — specific entropy
  • subscript \(f\) — saturated liquid state
  • subscript \(g\) — saturated vapor state
